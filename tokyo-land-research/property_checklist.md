# 物件チェックリスト（property_checklist.md）

候補物件が出たときに、上から順に埋めていくための確認リスト。
「埋まらない欄がある＝確認できていないリスク」と考える。空欄が多いほど判断は保留にする。

---

## 0. 基本情報

| 項目 | 記入欄 | メモ |
|---|---|---|
| 所在地（住居表示・地番） | | 地番は登記簿・公図で確認 |
| 最寄駅 / 路線 | | 複数路線あれば全部 |
| 駅距離（徒歩・分） | | 実測で歩くと印象が変わる |
| 価格 | | 売出価格。指値余地も別途メモ |
| 土地面積（㎡ / 坪） | | 登記面積と実測面積の差に注意 |
| 建物面積（㎡） | | 古家ありの場合 |
| 坪単価（土地） | | 価格 ÷ 坪数。周辺相場と比較 |
| 種別 | | 土地 / 古家付き土地 / 戸建て 等 |

## 1. 権利関係

| 項目 | 記入欄 | メモ |
|---|---|---|
| 所有権 or 借地権 | | 借地なら旧法/普通/定期の別 |

| 底地か | | 底地のみは収益＝地代のみ |

| 抵当権・根抵当権 | | 決済時に抹消されるか |
| 差押え・仮差押え | | 滞納・訴訟の痕跡 |
| 仮登記・買戻特約 | | 残ると厄介 |
| 地上権・地役権・賃借権 | | 第三者の利用権 |
| 境界確定の有無（確定測量図） | | 未確定はトラブルの種 |
| 越境（こちらへ/相手へ） | | 越境物の有無と覚書 |
| 私道負担の有無・面積 | | 有効宅地が減る |

## 6. 資金・出口

| 項目 | 記入欄 | メモ |
