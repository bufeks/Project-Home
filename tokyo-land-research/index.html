② なぜ安いのか／なぜ都心では安くなりにくいのか #

安くなる仕組み

欠陥が「最も価値のある権利＝（再）建築して売れる建物を持つ権利」を制限・喪失させるため。

  • 再建築不可：滅失しても建て直せない
  • 借地権：土地を所有できない（地代・承諾料が続く）
  • 底地：収益が低い地代のみ
  • 共有持分：単独では使えず売れない
  • 低い評価額 → 固定資産税も低いが融資も付きにくい

融資が付きにくい根本理由

  • 担保価値が低い：銀行が処分しても安くしか売れない
  • 滅失再建リスク：再建築不可は火災・地震後に建て直せず延滞リスク
  • 借り手属性：激安物件の買い手は自己資金が薄いことが多い

→ 住宅ローン不可が多く、現金・ノンバンク・リフォームローンが実態。

都心で安くなりにくい理由：土地需要が厚く、欠陥があっても立地プレミアムで価格が下がりきらない（人気区の再建築不可でも割安性が消えることがある）。

必須の基礎概念

接道義務（建基法43条1項）

敷地は幅員4m以上の建築基準法上の道路に2m以上接する必要。未達は再建築不可。救済の43条但し書き（現43条2項2号許可）は個別審査・再建築のたび再申請で承継保証なし。

セットバック（42条2項道路）

幅員4m未満の「みなし道路」では中心線から2m後退。後退部分は建築不可・容積率/建ぺい率に算入不可 → 有効宅地が減る。

借地権の費用

地代（継続）／更新料／譲渡承諾料（借地権価格の約1割目安）／建替承諾料。旧法(1992/8前)は借地人有利、定期借地は更新なし更地返還。

私道持分・掘削承諾

私道接道はインフラ更新に掘削承諾が必要。持分・承諾がないとリフォーム・融資が詰まる頻出の隠れ瑕疵。

④ 目的別の現実的な買い方 #

1. 自宅用

所有権の古家付き土地／狭小戸建て／(条件次第で)借地権戸建て／再建築可能な小規模土地。

再建築不可は慎重。融資可否が最重要。将来売れるか・生活インフラ。

2. 小さな店舗用

古家付き土地／商店街近くの小規模戸建て／路面の借地権建物／改修可能な再建築不可／定期借地・定期借家。

用途地域・飲食許可・保健所・消防・排気/臭気/音・近隣・搬入出。

3. アトリエ・作業場用

古家／倉庫／工房利用可の戸建て／再建築不可の改修利用／準工業地域周辺。

音・匂い・火気・電気容量・換気・搬入出・近隣苦情。

4. ギャラリー・文化拠点用

古家付き土地／再建築不可／路地奥／借地権戸建て／定期借家／公有地・空き家活用制度。

集客動線・近隣・消防避難・イベント可否・収益化の難しさ。

5. 投資用

再建築可能な古家付き土地／賃貸可能な戸建て／(出口理解のうえ)借地権・狭小／任意売却。

利回りで判断しない。出口戦略が最重要。次の買主も融資を使えるか。

6. 将来の土地保有用

狭小地／隣地買収余地のある土地／低利用地／古家付き／接道改善余地のある土地。

保有コスト・固定資産税・すぐ使えないリスク・隣地交渉の不確実性。

⑤ 想定リスク一覧 #

1. 法的・権利関係

  • 借地権／底地／共有持分のみ
  • 抵当権・差押え・仮登記・地上権・賃借権
  • 境界未確定・越境
  • 私道持分なし・通行/掘削承諾なし
  • 再建築不可・接道義務未達

確認：司法書士・不動産業者・(必要に応じ)弁護士

2. 建築・利用

  • 建替不可・改修のみ
  • 用途地域で店舗/工房が制限
  • 建ぺい/容積が低い・防火地域でコスト増
  • セットバックで面積減
  • 解体/造成/擁壁・アスベスト/埋設物/土壌汚染

確認：建築士・工務店・解体業者・土地家屋調査士・建築指導課

3. 金融・資金調達

  • 競売/公売は住宅ローン不可
  • 再建築不可/借地/底地/共有は融資困難
  • 現金決済・短い納付期限
  • 借り換え・売却が難しい（出口が狭い）

確認：金融機関・不動産業者・税理士・FP

4. 公売・競売特有

  • 内覧不可・占有者・残置物
  • 滞納・現況と資料の差異
  • 契約不適合責任を追及しにくい
  • プロとの競争で高値掴み

確認：競売に強い不動産業者・司法書士・弁護士・建築士

近隣・運用リスク：近隣関係・騒音/臭気/人の出入りへの反対・私道承諾・用途と地域の不一致・搬入出・消防避難・保健所許可。「使える土地」と「買える土地」は違う。安くても運用できなければ意味がない。

⑥ 初心者向けリスク判定（A〜E） #

判定意味
A初心者でも検討可能。ただし専門家確認は必須
B条件次第で検討可能。不動産業者・建築士・司法書士の確認が必要
C初心者にはやや難しい。学習対象として有効。購入は専門家同伴が前提
D初心者が購入するには難しい。なぜ難しいかを学ぶ価値はある
E原則 購入対象外。リスク構造の学習用サンプルとして扱う
D/Eは単に除外しない。「なぜD/Eか／最大リスク／確認すべき専門家／何が整えばC/Bに上がるか／教材として何を見るか」を必ず記録する。

原則避ける（→D/E）

共有持分のみ／底地のみ／借地複雑／占有者あり／退去交渉要／再建築不可かつ建物劣悪／接道不明／私道持分なし／境界未確定／越境／極端な狭小／融資不可／用途未定／出口なし／内覧不可かつ建物不明／価格以外の魅力なし。

比較的扱いやすい（条件次第A/B）

所有権／接道明確／再建築可／解体費が読める古家付き／境界確定／抵当権が決済時抹消／用途地域と用途が一致／融資の可能性／地場業者が介在／建築士に事前相談可／出口複数／安い理由を説明できる。

⑨ 学習用サンプル（買わずに力をつける） #

購入候補としてD/Eでも、安い理由とリスク構造を学ぶ教材として残す。各サンプルは「なぜ安い／どこが危ない／どの資料で分かる／専門家の着眼点／どんな人なら買えるか／何が整えば検討対象に上がるか」を整理。

再建築不可

接道義務未達で建替不可。価格より先に接道を見る癖を。43条但し書き or 隣地一体化で改善余地。

借地権

土地は地主のもの。地代・更新料・譲渡承諾料が続く。旧法/普通/定期の別と残存期間・承諾の取りやすさを総コストで。

底地

借地人がいて自由に使えない。完全所有権化(借地権買取/借地人へ売却)ができないと出口が極端に狭い。

共有持分のみ（原則回避）

単独で使えず売れず、分割請求は訴訟化。統合できる資力・交渉力がある人のみ。

古家付き土地（B寄り）

解体費・セットバック減・インフラ更新を入れて割安性が残るか。接道・境界が明確なら入口になりうる。

競売 / 公売

BITの三点セットを月に数件、買わずに精読するだけでリスク構造が体系的に学べる。学習価値最大。

⑪ 最終判断ルール・撤退条件 #

「買い」と判定する5条件（すべて必須）： ①安い理由を言語化できる ②リスクを専門家に確認できる ③買った後の使い道がある ④出口が少なくとも1つある ⑤諸費用を含めても割安性が残る。 1つでも欠ければ「見送り」または「要追加調査」。
撤退条件（いずれかに該当で原則中止）： 再建築可否/接道/境界が確認できない・融資も現金も難しい・解体改修込みで割安でない・用途が実現できない・占有者/共有者との交渉が必要・権利関係を説明されても理解できない・出口戦略がない・安い理由が説明できない・価格だけで欲しくなっている・専門家確認なしで入札申込しようとしている。
→ 撤退時も学習サンプルとして「撤退理由・主要リスク・追加確認で再検討できるか・次回の注意点」を記録する。
