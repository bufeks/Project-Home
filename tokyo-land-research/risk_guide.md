# リスクガイド（risk_guide.md）

都心の安い土地・古家付き土地・借地権・再建築不可・公売・競売を検討するときの想定リスクと、
「誰に何を確認するか」「初心者が何を学ぶか」をまとめた教材。

安い物件には必ず理由がある。本ガイドは「危険だから避ける」ためではなく、
**安い理由を構造として理解し、扱えるリスクと扱えないリスクを切り分ける**ためのもの。

---

## 1. 法的・権利関係リスク

### 主なリスク
- 所有権ではなく**借地権**（土地は他人のもの。地代・更新料・各種承諾料が発生）
- **底地**で、借地人がいて自由に使えない（収益は低い地代のみ）
- **共有持分のみ**の売却 → 単独で使えず、売れず、できるのは分割請求か持分売却だけ
- 抵当権・差押え・仮登記・地上権・賃借権が付いている
- **境界未確定**（確定測量図がない）
- **越境**（隣家の屋根・塀・木・配管がこちらに、またはこちらが越境）
- **私道持分なし** → 通行・掘削で隣地承諾が必要
- **通行・掘削承諾なし** → インフラ更新ができず、リフォーム・融資が詰まる
- **再建築不可**（建築基準法の接道義務を満たさない）
- 借地契約の内容・残存期間・各種承諾料が不明

### 確認すべき専門家
司法書士（登記・権利）／不動産業者（実務・相場）／必要に応じて弁護士（共有・借地紛争）

### 学習ポイント
- 所有権と借地権の違い、旧法/普通/定期借地の違い
- 底地と借地は表裏一体（どちらも単独では弱い）
- 共有持分の難しさ（持っているのは「揉める権利」）
- 登記簿（甲区＝所有権、乙区＝担保）の読み方
- **接道義務**：建築基準法43条1項。敷地は幅員4m以上の建築基準法上の道路に2m以上接する必要。満たさないと再建築不可
- **43条但し書き（現43条2項2号許可）**：救済許可だが個別審査・再建築のたび再申請・承継保証なし
- 私道持分と掘削承諾がなぜ致命的になりうるか

## 2. 建築・利用リスク

### 主なリスク
- 建て替えできない（再建築不可）→ 改修のみ
- 用途地域により店舗・飲食・宿泊・工房利用が制限される
- 建ぺい率・容積率が低く、想定より小さい建物しか建たない
- **防火・準防火地域**で建築コストが上がる
- 高度地区・日影規制・斜線制限が厳しい
- **セットバック**で有効面積が減る（42条2項道路は中心線から2m後退、後退部分は建築不可・容積算入不可）
- 狭小・変形・旗竿で設計自由度が低い
- インフラ引込に追加費用（古い水道管・ガス未引込・下水未接続・電気容量不足）
- 解体費・造成費・擁壁補修費が高額
- アスベスト・地中埋設物・土壌汚染の可能性

### 確認すべき専門家
建築士／工務店／解体業者／土地家屋調査士／行政の建築指導課

### 学習ポイント
- 再建築可否の見方（接道・道路種別・幅員）
- 建ぺい率・容積率から逆算する「建てられる箱」
- 用途地域ごとの店舗・工房の可否
- セットバックで土地がどれだけ削られるか
- 「改修できる建物」と「できない建物」の違い
- 建築コストが上がる要因（防火・狭小・接道難）

## 3. 金融・資金調達リスク

### 主なリスク
- 競売・公売は**住宅ローンが使いにくい**（決済期限が短い）
- 再建築不可・借地権・底地・共有持分は**融資がつきにくい**（担保評価が低い）
- 現金決済が必要になりがち
- 落札後の支払い期限が短い／入札保証金が必要
- 想定外の解体・測量・登記・税・改修費が発生
- 将来の借り換え・売却が難しい
- **次の買主も融資を使えず、出口が狭くなる**

### 融資がつきにくい根本理由（学習）
1. **担保価値が低い** … 銀行が競売処分しても安くしか売れない → 担保に取らない
2. **滅失再建築リスク** … 再建築不可の建物が火災・地震で失われると建て直せず、ローン＋家賃の二重負担で延滞リスク
3. **借り手属性** … 激安物件の買い手は自己資金が薄いことが多く審査に通りにくい
→ 住宅ローン不可が多く、現金・ノンバンク（高金利）・リフォームローンで対応するのが実態

### 確認すべき専門家
金融機関／不動産業者／税理士／FP

### 学習ポイント
- 住宅ローンと事業用融資の違い、担保評価とは何か
- 融資がつきにくい物件の特徴
- 物件価格以外に必要な諸費用（仲介・登記・税・測量・解体・改修）
- 出口で融資がつかないリスク

## 4. 公売・競売特有のリスク

### 主なリスク
- 内覧できない場合がある
- **占有者**がいる → 明渡し交渉が必要
- 残置物がある
- 管理費・修繕積立金の滞納
- 物件情報が限定的（現況と資料が異なることがある）
- 落札後のキャンセルが難しい
- **契約不適合責任を追及しにくい**（瑕疵があっても自己責任）
- プロ業者との競争で価格が上がりやすい
- 支払い期限が短い
- 入札価格の判断を誤ると高値掴み

### 公売と競売の違い（学習）
- **競売**：裁判所（民事執行）が実施。BITで三点セット（物件明細書・現況調査報告書・評価書）が公開され情報量が比較的多い
- **公売**：税金滞納に基づき税務署・自治体が実施。情報は競売より限定的なことが多い

### 確認すべき専門家
競売に詳しい不動産業者／司法書士／弁護士／建築士

### 学習ポイント
- 三点セットの読み方
- 売却基準価額と買受可能価額の意味（買受可能価額＝売却基準価額の8割）
- 占有者リスク・明渡し交渉の難しさ
- 入札価格の決め方（相場−リスク−諸費用−利益）

## 5. 近隣・運用リスク

### 主なリスク
- 近隣関係が悪い／騒音・臭気・人の出入りへの反対
- 私道利用で近隣承諾が必要
- 飲食・工房・ギャラリーが地域と合わない
- 搬入出が難しい・前面道路が狭い
- ゴミ出し・駐輪・駐車の問題
- イベント利用に向かない／消防・避難条件を満たせない
- 保健所の営業許可が取れない

### 確認すべき専門家・窓口
不動産業者／建築士／行政／保健所／消防署／近隣ヒアリング

### 学習ポイント
- 不動産は土地そのものだけでなく周辺環境込みで見る
- 店舗・文化拠点は法規だけでなく近隣関係が成否を分ける
- **「使える土地」と「買える土地」は違う**
- 安くても運用できなければ意味がない

## まとめ：リスクの切り分け方

| 扱いやすい（条件次第でA/B） | 扱いにくい（原則D/E） |
|---|---|
| 解体費が読める古家付き | 共有持分のみ／底地のみ |
| 境界確定済み | 境界未確定・越境未解決 |
| 接道明確・再建築可 | 接道不明・再建築不可で建物劣悪 |
| 抵当権が決済時抹消 | 占有者あり・明渡し交渉要 |
| 用途と用途地域が一致 | 用途が決まらない／実現不可 |
| 融資 or 現金の目処あり | 融資不可かつ現金も厳しい |

**原則**：危険だから捨てるのではなく、「危険なので購入は避けるが、教材として残す」。
A/B（買い候補）と C/D/E（学習価値）は分けて管理する。
